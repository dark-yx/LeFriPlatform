## ğŸš€ Getting Started

### Prerequisites
- Node.js (Latest LTS version)
- MongoDB

### Installation

1. Clone the repository
```bash
git clone https://github.com/dark-yx/LeFriPlatform.git
```

2. Install dependencies

```bash
cp .env.example .env
```

4. Start the development server
